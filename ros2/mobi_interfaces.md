## Einleitung

## Installation

1. In einen ROS Workspace wechseln

   ```bash
   cd ~/ros2_ws
   ```

2. Interfaces herunterladen

   ```bash
   git clone https://github.com/FHWN-Robotik/mobi_interfaces.git src/mobi_interfaces
   ```

3. Workspace builden

   ```bash
   colcon build
   ```
